TikTok Creative Studio 自动化

自动生成 TikTok 广告视频 — 填写产品信息，一键批量生成

首次使用

  1. 双击 start.bat 启动（会自动关闭并重启 Chrome）
  2. 在弹出的 Chrome 中登录 TikTok Creative Studio
  3. 打开 http://localhost:8000 开始使用

操作流程

  1. 填写产品信息、上传图片
  2. 选择类目和生成数量
  3. 点击「预览提示词」确认内容
  4. 点击「确认并执行」开始自动生成
  5. 系统自动操作 TikTok 页面，生成完毕后可下载视频

注意事项

  • Chrome 必须通过 start.bat 启动（带调试端口）
  • 执行期间不要手动操作 TikTok 页面
  • 每次生成需要 5-15 分钟，请耐心等待
  • 如果出错，刷新页面重新提交即可
